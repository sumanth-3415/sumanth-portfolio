# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website developed to showcase my skills, education, projects, and achievements in an attractive and professional manner.

## 🚀 About the Project

This portfolio website serves as a digital representation of my academic journey, technical skills, and project experience. It is designed with a clean user interface, responsive layout, and interactive elements to provide visitors with a smooth browsing experience.

The website highlights my background, technical expertise, featured projects, educational qualifications, and contact information in a well-structured format.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern glassmorphism-inspired user interface
* Interactive navigation menu
* Animated typing effect in the hero section
* About Me section
* Technical Skills showcase
* Featured Projects section
* Education timeline
* Resume download functionality
* Contact section with social media links
* Smooth scrolling and hover animations
* Professional dark theme design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts
* Typed.js

## 📸 Sections Included

### Home

Introduces the portfolio owner with a professional landing page and animated text effects.

### About

Provides a brief overview of educational background, interests, and career objectives.

### Skills

Displays technical skills and technologies.

### Projects

Highlights major academic and personal projects.

### Education

Showcases educational qualifications in a timeline format.

### Contact

Provides direct access to email, phone, GitHub, and LinkedIn profiles.

## 🎯 Purpose

The primary goal of this portfolio is to create a professional online presence that can be shared with recruiters, internship providers, and industry professionals while demonstrating technical and development skills.
